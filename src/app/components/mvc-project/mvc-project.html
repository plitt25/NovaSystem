Descripción del Proyecto

Sistema web desarrollado bajo el patrón MVC (Modelo-Vista-Controlador), con arquitectura cliente-servidor moderna que integra NestJS en el backend, PostgreSQL como base de datos y Angular en el frontend.

Arquitectura y Funcionamiento

El sistema sigue el modelo MVC, donde cada capa tiene responsabilidades claramente definidas:

  • Modelo (PostgreSQL + Prisma): Gestiona la persistencia de los datos y su estructura mediante un ORM eficiente.
  • Controlador (NestJS): Expone endpoints RESTful, manejando las peticiones HTTP y coordinando la lógica del negocio.
  • Vista (Angular): Consume los servicios del backend a través de HTTP, ofreciendo una interfaz interactiva y amigable para el usuario.
